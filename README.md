## Chatbot UI Features
The ChatGPT-style UI includes:
- Chat-style conversation layout
- Typing animation for AI responses
- Markdown support (headings, lists, code blocks)
- Syntax highlighting for code
- Copy button for code blocks
- Light / Dark theme toggle
- File upload (image / PDF preview)
- **Delete chat option** to clear all conversations

## About AI (Important)
During development, this project was **fully connected to the Google Gemini API** and was working properly with real AI responses.

Later, I removed the API key from the frontend **for security and billing safety reasons**, because:
- API keys should not be public
- Public repositories can leak keys
- API misuse can cause unexpected billing
